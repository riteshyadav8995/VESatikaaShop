# VESatikaa Collections

A modern, premium e-commerce portfolio and catalog website for a fashion boutique. Built with React and Vite.

## 🌟 Features

- **Responsive Design**: Flawless experience across mobile, tablet, and desktop devices.
- **Dynamic Collections**: Filter products by categories like Women's, Men's, Summer, Winter, Accessories, and New Arrivals.
- **Dual Inquiry System**: 
  - **WhatsApp Integration**: Instant, pre-filled WhatsApp messages for product-specific inquiries.
  - **Contact Form**: Built-in email contact form powered by Formspree.
- **Modern UI**: Smooth animations, auto-scroll behaviors, and a sleek dark-themed footer with glassmorphism touches.

## 🚀 Tech Stack

- **Frontend Framework**: React.js 
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (CSS Variables, Flexbox, Grid)

## 🛠️ Local Setup

Follow these steps to run the project locally on your machine:

1. **Install Dependencies**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Start the Development Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser to view the app.

3. **Build for Production**
   To create an optimized production build (for Netlify, Vercel, etc.):
   ```bash
   npm run build
   ```
   *Note: A `_redirects` file is included in the `public` folder to ensure React Router works correctly on Netlify.*

## 📬 Contact & Inquiries
All contact form submissions are routed securely via Formspree. Direct inquiries are routed to the store's designated WhatsApp number.
